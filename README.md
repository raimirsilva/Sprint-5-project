# Sprint-5-project
## Dashboard web interativo para análise de dados de vendas de carros
### Painel de controle onde usuários podem explorar informações sobre veículos de forma visual e interativa.

Aplicativo web desenvolvido com Streamlit para análise de dados de vendas de carros:
[Acesse o dashboard aqui](https://sprint-5-project-8vqs.onrender.com)

Funcionalidades principais:

1. Visualização de dados:
- Histograma: Mostra distribuição de dados (ex: quantos carros por faixa de quilometragem)
- Gráfico de dispersão: Mostra relação entre duas variáveis (ex: preço vs quilometragem)

2. Interatividade:
- Botões: Usuário clica para gerar gráficos
- Caixas de seleção: Usuário escolhe que tipo de análise quer ver

3. Interface amigável:
- Cabeçalhos: Explicam o que o usuário está vendo
- Navegação simples: Fácil de usar, mesmo para não-programadores
